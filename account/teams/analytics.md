# Аналитика команды

## Основные метрики
1. **Активность ИИ**:
   - Количество запросов
   - Популярные функции
   - Среднее время ответа
2. **Продуктивность**:
   - Сгенерированный код
   - Рефакторинг
   - Автоматизированные задачи
3. **Использование**:
   - Активные пользователи
   - Пиковые часы работы
   - Использование кредитов

## Доступные отчеты
- Ежедневная сводка
- Недельные тренды
- Индивидуальная статистика
- Сравнение между командами

## Экспорт данных
Данные можно экспортировать в:
1. CSV
2. JSON
3. Google Sheets (автоматическая синхронизация)

## Настройка уведомлений
Установите пороговые значения для:
- Расхода кредитов
- Необычной активности
- Достижения лимитов

Для доступа к расширенной аналитике требуется роль администратора.
